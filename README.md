# codsoft_internship-student-management-system-
**Overview**
Developed a Student Management System using Python.
Manages student records efficiently through a menu-driven interface.
Allows users to add, view, search, update, and delete student details.
Stores student data permanently using CSV file handling.
Implements Object-Oriented Programming (OOP) concepts using a Student class.
Utilizes Python fundamentals such as variables, loops, conditional statements, and functions.
Supports searching and sorting operations on student records.
Demonstrates CRUD (Create, Read, Update, Delete) operations.
Provides hands-on experience with file handling and data management.
Suitable as a beginner-level mini project for academic and practical learning.
